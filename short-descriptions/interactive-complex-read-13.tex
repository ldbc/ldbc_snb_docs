\paragraph{Q13.}
Given two Persons, find the shortest path between these two Persons in
the subgraph induced by the Knows relationships.
Return the length of this path:
\begin{itemize}
\tightlist
\item
  \(-1\) : no path found
\item
  \(0\): start person = end person
\item
  \(> 0\): regular case
\end{itemize}
